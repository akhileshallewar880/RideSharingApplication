### Legend:
- **в”Ҹв”Ғв”Ғв”Ғв”“** = Start/End stop (square marker)
- **в—Ҹ в—Ҹ в—Ҹ** = Intermediate stop (circle marker), passed
- **в—Ӣ в—Ӣ в—Ӣ** = Intermediate stop (circle marker), upcoming
- **рҹҡ—** = Current position (vehicle icon)
- **вң“** = Check mark for passed stops
- **в”Ӯ в”Ӯв”Ӯ в”Ӯ** = Timeline connection (green = passed, grey = upcoming)
- **рҹ”ј** = Pickup indicator (green)
- **рҹ”Ҫ** = Dropoff indicator (red)
- **рҹ”ё** = Current stop badge (orange)

## рҹҺҜ Use Cases

### 1. Scheduled Ride
```
Status: SCHEDULED

No current stop indicator
No arrival times
```

### 2. In-Progress Ride
```
Status: IN PROGRESS
Some stops: Green (passed)
One stop: Orange (current)
Remaining: Grey (upcoming)
Some arrival times shown
```

### 3. Completed Ride
```
Status: COMPLETED
All stops: Green (passed)
No current stop
All arrival times shown
```

---

## рҹ’Ў Tips for Reading the Timeline

1. **Follow the line**: Green = already traveled, Grey = yet to come
2. **Check the badge**: Orange "CURRENT" shows where driver is now
3. **Count passengers**: Numbers show pickups/drops at each stop
4. **Compare times**: Scheduled vs Actual shows if on time
5. **Watch distances**: Track progress by kilometer markers
